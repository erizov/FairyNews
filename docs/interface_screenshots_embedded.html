Скриншоты интерфейса

Документ самодостаточный: изображения встроены в файл (data URI), внешние ссылки на картинки не нужны.

Шаг 1 — новость

На первом экране вводится или выбирается новость — исходный текст для сказки.

Шаг 2 — пресет

Второй шаг задаёт стиль опоры для поиска похожих сюжетов (в примере — русский фольклор).

Шаг 3 — сказка и вопрос

После запуска показываются сказка, сводка по этапам, вопрос по тексту и ответ для сверки.

Сводка отчётов

Страница со списком сохранённых прогонов: время, объём текста, ссылка на детали.

Детали прогона

Развёрнутая карточка прогона: подсказки из индекса, оценки согласованности, полный текст.
